## Java基础

### 1、Java的基本数据类型和引用数据类型有哪些？

- 基本数据类型

  **基本数据类型一共有八种：**

  1. 整数类型 ：int 、short、byte、long

  2. 浮点类型 ： double、float

  3. 字符类型 ： char

  4. 布尔类型 ：boolean

     ![image-20220320144406706](https://cdn.jsdelivr.net/gh/gaoziman/bed/img/image-20220320144406706.png)

- 引用数据类型

  ​	**非基本数据类型都是引用数据类型**

  ​	**引用数据类型大体上一共有5种**

  1. 类
     1. String类（String）
     2. 封装类
     3. 包装类（integer 等）
  2. 接口
  3. 数组
  4. 枚举
  5. 注解
